Dom's Burger Joint

https://doms-burger-joint.herokuapp.com/

https://github.com/Domchap90/Doms_burger_joint

A full stack website for a typical online food delivery service. This includes a database of the food items which is hosted by sqlite. Orders can be created from the database & other CRUD operations are available on the food items.

The site features a fully secure checkout (Stripe payments API), that utilises webhooks and email confirmation for the user's successful orders. It also verifies the customer's postcode is valid by checking the distance from store to the customer's address via google maps distance matrix API.

The app also gives the user an option to sign up as a member and benefit from a reward scheme, consisting of adding a free burger to the user's order after every 5th order made. The login is configured by the django allauth app.

Recomovie

https://recomedia.herokuapp.com/

https://github.com/Domchap90/recoMedia

A data-centric dynamic app for users to build profiles and share their favourite movies through ratings and reviews given. User profiles are visible and if logged in can be fully edited through CRUD operations on reviews.

Basic Login - plain text password storage. Data stored in mongoDB (Atlas) and queries performed via Python Flask control. Connects to OMDB API for film validation.
